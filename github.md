# Bienvenida a GH-900

## Introducción
*GitHub* es una plataforma líder para la colaboración en desarrollo de software, que permite a equipos y comunidades trabajar juntos de manera eficiente.

**Material del curso:**
- Fundamentos de Git y GitHub
- Colaboración y administración de proyectos
- Seguridad y buenas prácticas
- Uso de herramientas modernas como GitHub Copilot y Codespaces

## Contenido

### Día 1
- Introducción a Git  
- Introducción a GitHub  
- Introducción a los productos de GitHub  
- Configuración del examen de código en GitHub  
- Introducción a GitHub Copilot  
- Código con GitHub Codespaces  
- Administración del trabajo con GitHub Projects  
- Comuníquese eficazmente en GitHub con Markdown  

### Día 2
- Contribución a un proyecto de código abierto en GitHub  
- Administración de un programa InnerSource mediante GitHub  
- Mantenimiento de un repositorio seguro mediante procedimientos recomendados de GitHub  
- Introducción a la administración de GitHub  
- Autenticación y autorización de identidades de usuario en GitHub  
- Administración de cambios de repositorio por medio de solicitudes de incorporación de cambios en GitHub  
- Organización del historial de un repositorio y búsqueda mediante GitHub  
- Uso de GitHub Copilot con Python
